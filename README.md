LOAN PREDICTION SYSTEM(End-To-End ML Project)

PROBLEM STATEMENT
This project predicts whether a loan will be approved or rejected based on applicant details.

TECH STACK
Python
Pandas,Numpy
Scikit-learn
Flask(Backend API)
Streamlit(Frontend UI)
MLFlow(Experiment Tracking)
Joblib(Model Saving)

WORKFLOW
1.Data Cleaning
2.Data Preprocessing
3.Encoding
4.Model Training
5.Experiment Tracking using MLflow
6.Model Saving using Joblib
7.Flask API for prediction
8.Streamlit UI for user interaction

ALGORITHM: Random Forest Classifier

OUTPUT: Loan Approved(1) / Loan Rejected(0)

How to Run:
1.Run app.py
2.Run stramlit_app.py
